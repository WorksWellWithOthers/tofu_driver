# Tofu Shop Scene Layers

This folder is reserved for optional transparent future overlay art for the parked-only Tofu Shop
Living Scene.

V1 visible scene presentation uses full-scene images from the parent folder, not separate visible
layer tiles. Only add overlay art here after the full-scene approach is proven.

Do not add raw prompts, maps, route traces, street names, GPS-derived imagery, or active-drive
reward art here.
