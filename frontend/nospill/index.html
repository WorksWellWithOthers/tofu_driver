Mount Orientation

Device sensors measure movement relative to the phone, so mount orientation matters.

Recommended: phone flat, screen up, top pointed toward the windshield.

If using an upright windshield mount, choose Upright so braking and acceleration are read from the correct axis.

Flat mount maps side-to-side from device X and braking/acceleration from device Y.

While parked or moving very slowly in a safe place, gently accelerate/brake and confirm which axis changes.
