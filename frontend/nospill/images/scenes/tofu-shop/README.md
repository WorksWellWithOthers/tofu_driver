# Tofu Shop Living Scene Asset Drop Folder

This folder is reserved for future parked-only Tofu Shop scene art.

Expected top-level file:

- `scene_tiny_shop_empty.webp`
- `scene_tiny_shop_working.webp`
- `scene_tiny_shop_upgraded.webp`
- `scene_busy_shop_established.webp`
- `scene_busy_shop_with_covered_car.webp`

The V1 visible UI uses one cohesive full-scene image at a time. Optional future overlay files may
live in `layers/`, and optional future small activity sprites may live in `sprites/`, but those
should not appear as separate placeholder panels in the Tofu Shop Overview.

Do not put raw prompts, source PSDs, unsafe driving imagery, maps, route traces, street names, or
location references here. Final assets should be static-browser friendly, cosmetic only, and safe
for parked Tofu Shop screens.
